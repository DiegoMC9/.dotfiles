# My Dotfiles
--------------------
## i3wm config
### Instalation
**STOW**
## WARNING: BACKUP AND DELETE YOUR CONFIG BEFORE STOWING
```
# common configs for both desktop and pc
stow common
# desktop config
stow i3-dsk
# pc config
stow i3-pc
```
